# Description
A vintage-inspired wavetable synthesizer.

# Features
- resampling
- downsampling
- soft/hard distortion
- basic hp/lp filter
- polyphony
  
# Requirements
- [JUCE](https://github.com/juce-framework/JUCE?tab=readme-ov-file)
